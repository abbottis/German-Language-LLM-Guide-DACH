# GERMAN STYLE – LOCALE: ÖSTERREICH (de-AT)

Dieses Modul ergänzt die Kernregeln für österreichisches Deutsch. Aktiviere es zusätzlich zu core.md, wenn der Text für ein österreichisches Publikum bestimmt ist.

## Wortschatz

Verwende die österreichische Variante, wenn sie die geläufigere ist:

| Bundesdeutsch | Österreichisch |
|---------------|----------------|
| Januar | Jänner |
| dieses Jahr | heuer |
| Abitur | Matura |
| Treppenhaus | Stiegenhaus |
| Quark | Topfen |
| Tomate | Paradeiser |
| Kartoffel | Erdapfel |
| Sahne | Obers |
| Brötchen | Semmel |
| Aubergine | Melanzani |
| Aprikose | Marille |
| Pfannkuchen | Palatschinke |
| lecker | gut (in Österreich wird „lecker" als bundesdeutsch empfunden) |
| Tüte | Sackerl |

## Grammatik

1. **Perfektbildung mit „sein"** – Österreichisches Deutsch verwendet „sein" statt „haben" bei Verben der Körperhaltung:
   - Bundesdeutsch: „Ich habe gestanden / gesessen / gelegen"
   - Österreichisch: „Ich bin gestanden / gesessen / gelegen"

2. **Diminutive auf -erl** – Wo angemessen, -erl statt -chen verwenden:
   - „Sackerl" statt „Säckchen"
   - „Häferl" statt „Tässchen"

3. **Artikel bei Eigennamen** – Im österreichischen Deutsch ist der bestimmte Artikel vor Vornamen üblich und kein Fehler:
   - „Der Thomas hat angerufen" ist stilistisch akzeptabel.

## Anrede & Ton

4. **Höflichkeitsform** – In Österreich wird die Höflichkeitsform tendenziell häufiger und länger beibehalten als in Deutschland. Im Zweifelsfall siezen.

## Institutionen & Begriffe

5. Verwende die korrekte österreichische Terminologie:
   - Bundesland (nicht „Land" im Sinne von Gliedstaat)
   - Bezirk (Verwaltungseinheit)
   - Gemeinde (nicht „Kommune")
   - Nationalrat / Bundesrat (nicht „Bundestag")
